Get Process Variables

Retrieves all variables of a given process instance.

Method

GET /process-instance/{id}/variables

Parameters

Path Parameters

Name Description
id The id of the process instance to retrieve the variables for.

Result

A json object with a single variables property, which holds an array of variable objects. Each variable object has the following properties:

Name Value Description
name String The name of the process variable.
value ?? The value of the Java object.
type String The class name of the variable object.

Response codes

Code Media type Description
200 application/json Request successful.
400 Process definition with given id does not exist.
