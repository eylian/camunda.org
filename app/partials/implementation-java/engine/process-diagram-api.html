Process Diagram API

A BPMN process diagram is a formidable place to visualize information around your process. Basically you have to options to do this in the camunda BPM stack:

  • The Java Process Diagram API (described below)
  • Our JavaScript BPMN Renderer (will be documented soon - you can see it in action throughout this website!)
When using the Process Diagram API you can deploy a PNG image together with your BPMN 2.0 Process Model. Then you have an easy API to query the image and normalized coordinates for the process model. With these informations you can easily visulize anything on the process model.
