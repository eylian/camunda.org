Process Diagram API

A BPMN process diagram is a formidable place to visualize information around your process. Basically you have to options to do this in the camunda BPM stack:
  • The Java Process Diagram API (described below)
  • Our BPMN Javascript libraries for parsing, executing and rendering BPMN 2.0 with Java Script

Process Diagram API

When using the Process Diagram API you can deploy a PNG image together with your BPMN 2.0 Process Model. Then you have an easy API to query the image and normalized coordinates for the process model. With these informations you can easily visulize anything on the process model. The following image shows an example using an BPMN 2.0 model from Adonis (see Roundtrip with other BPMN 2.0 Modelers): See Process Diagram API for detailed information on the usage.

BPMN Javascript libraries

The camunda BPMN Javascript libraries for parsing, executing and rendering BPMN 2.0 with Java Script are not yet documented here, check out camunda-bpmn.js on GitHub for more information.

Note that all BPMN diagrams on www.camunda.org are rendered with this library.
