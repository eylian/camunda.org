Delegate Task

Delegate a task to another user.

Method

POST /task/{id}/delegate

Parameters

Path Parameters

Name Description
id The id of the task to delegate.

Request Body

A json object with the following properties:

Name Description
userId The id of the user that the task should be delegated to.

Result

This method returns no content.

Response codes

Code Media type Description
204 Request successful.
500 If the task does not exist or delegation was not successful.

Example

Request

POST /task/anId/delegate

Request body:

Response

Status 204. No content.
